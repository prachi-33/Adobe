# adobe-addon

This repository contains the Adobe Creative Cloud Web Add-on code (converted from the previous "design-to-code" project). It includes the add-on source under `src/` and backend utilities under `backend/`.

To create a remote GitHub repository and push changes:

1. Create a repo on GitHub named `adobe-addon` under your account.
2. Locally:

```bash
git remote add origin https://github.com/<your-username>/adobe-addon.git
git branch -M main
git push -u origin main
```

Replace `<your-username>` with your GitHub username.
## About

This project has been created with _@adobe/create-ccweb-add-on_. As an example, this Add-on demonstrates how to get started with Add-on development using React and TypeScript with Document Sandbox Runtime.

## Tools

-   HTML
-   CSS
-   React
-   TypeScript

## Setup

1. To install the dependencies, run `npm install`.
2. To build the application, run `npm run build`.
3. To start the application, run `npm run start`.
